# North Star Metric: Habit Rabbit

**날짜**: 2026-03-07
**비즈니스 게임**: Productivity (Attention 요소 결합)

---

## North Star Metric

> **주간 습관 완수자 수 (Weekly Habit Completers)**
>
> 지난 7일 동안 자신이 설정한 습관의 **80% 이상을 완료한 사용자 수**

**목표**: 출시 3개월 내 MAU의 35%

### 왜 이 메트릭인가

- **체크박스 역전 방지**: 앱 사용 횟수가 아닌 실제 습관 달성 여부를 측정
- **스트릭 스트레스 해소**: 연속일(스트릭)이 아닌 주간 달성률로 하루 쉬어도 포함 가능
- **비즈니스 선행 지표**: Weekly Completers ↑ → D30 리텐션 ↑ → 구독 전환 가능성 ↑

### 80% 기준 근거

| 기준    | 문제                                  |
| ------- | ------------------------------------- |
| 100%    | 완벽주의 촉진, 스트릭 스트레스와 동일 |
| **80%** | "5일 중 4일" — 현실적이고 지속 가능   |
| 50%     | 기준 너무 낮아 습관 형성 여부 불명확  |
