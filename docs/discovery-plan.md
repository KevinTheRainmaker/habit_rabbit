# Discovery Plan: Habit Rabbit

**날짜**: 2026-03-07
**제품 단계**: 신규 제품 (개발 전)
**디스커버리 질문**: 습관 트래커 포화 시장에서 당근 경제 + 토끼 컨셉 + AI 코칭의 조합으로 수익화 가능한 제품을 만들 수 있는가?

---

## 제품 개요

### 프리미엄 모델

- **무료**: 습관 3개, 기본 알림, 기본 통계, 당근 포인트 적립, 기본 아이템
- **유료**: 무제한 습관, 알림 세분화, 실패 패턴 분석, AI 코칭, 심화 통계, 스트릭 복구권(월 1개 적립), 프리미엄 아이템
- **공통**: 아이템 영구 소유, 일반 아이템 80% 환매, 한정 아이템 환매 불가
- **MVP 제외**: 가족/팀 스페이스, 현금 IAP

### 디자인 컨셉

토끼굴 속 토끼들. 습관 건강도에 따라 굴이 변화. 당근 포인트로 토끼/굴 꾸미기 아이템 구매.

### 기술 스택 방향

- 구독 관리: RevenueCat
- 개발 형태: 1인 개발, 수익화 목표

| E1  | 아이템 꾸미기 동기 | 랜딩 페이지 사전 등록      | 2주 내 사전 등록 200명        | 낮음 | 개발 시작 전  |
| E2  | 업셀 타이밍        | 베타 사용자 전환율 측정    | 복구권 사용자 15% 7일 내 전환 | 낮음 | 베타 2-4주    |
| E3  | AI 코칭 품질       | 컨시어지 MVP (수동 리포트) | 사용자 80% "유용" 평가        | 낮음 | 베타 2주 후   |
| E4  | 전환율 달성        | RevenueCat A/B 테스트      | 어느 가격대에서 2% 이상 전환  | 낮음 | 정식 출시 4주 |

### 실험 상세

#### E1: 랜딩 페이지 사전 등록 테스트

- **가설**: 토끼굴 꾸미기 게임 요소를 강조한 앱 소개 페이지를 보면, 20%는 사전 등록한다
- **실행**: 토끼굴 꾸미기 스크린샷/영상 포함 랜딩 페이지 → 소셜 공유 → 사전 등록 수 측정
- **성공**: 2주 내 200명
- **실패 시**: AI 코칭/분석 기능을 핵심 USP로 피벗

#### E2: 업셀 타이밍 베타 테스트

- **가설**: 스트릭이 처음 깨진 직후 복구권 제안 사용자는 미제안 사용자보다 전환 의향 2배 이상
- **실행**: 첫 실패 후 복구권 무료 1회 제공 → "유료 플랜에서 매달 받을 수 있어요" 안내 → 전환율 측정
- **성공**: 복구권 무료 사용자 중 15% 이상 7일 내 유료 전환
- **실패 시**: 3개 습관 한도 도달 시점을 주요 업셀 모먼트로 전환

#### E3: AI 코칭 컨시어지 MVP

- **가설**: 2주치 데이터 기반 분석 리포트를 받은 사용자의 80%는 "유용하다"고 평가한다
- **실행**: 베타 사용자 10명에게 수동으로 분석 리포트 작성 제공 → 반응 측정 후 자동화
- **성공**: "유료로 쓸 만하다" 응답 80% + 구체적 행동 변화 언급
- **실패 시**: AI 코칭 출시 시점을 데이터 1개월 축적 후로 지연. MVP 제외 검토

#### E4: RevenueCat 페이월 A/B 테스트

- **가설**: 복구권 + AI 코칭 + 무제한 습관 번들이 특정 가격에서 2% 이상 전환율
- **실행**: RevenueCat으로 2가지 가격 포인트 동시 테스트 (예: 3,900원 vs 6,900원/월)
- **성공**: 어느 한 가격에서 2% 이상 전환율
- **실패 시**: 프리미엄 기능 구성 재검토. 복구권 단품 IAP 등 대안 검토

## 의사결정 프레임워크

- E1 성공 (200명 사전 등록) → 개발 시작, 아이템 경제 MVP에 포함
- E1 실패 → AI 코칭을 핵심 USP로 재포지셔닝. 아이템 경제 규모 축소 검토
- E3 실패 → AI 코칭 MVP 제외, 실패 패턴 분석(통계)만 먼저 출시
- E4 실패 → 단품 IAP (복구권 팩, 당근 팩) 수익 모델 추가 검토

## MVP 기능 범위 권고

### 반드시 포함 (Core)

- 습관 등록/체크 (무료 3개 / 유료 무제한)
- 당근 포인트 + 스트릭 보너스
- 기본 아이템 샵 + 토끼/굴 꾸미기
- 스트릭 복구권 (유료 월 1개)
- 기본 알림 (유료: 세분화)
- 기본 통계 (유료: 심화)
- RevenueCat 구독 관리
- 온보딩 퀴즈 + "1개부터 시작" 권장

### 포함 검토 (E1/E3 결과 후 결정)

- AI 코칭 (E3 검증 후)
- 실패 패턴 분석 (데이터 2주 축적 후 활성화)
- 홈/잠금화면 위젯

### MVP 제외 (이후 단계)

- 가족/팀 스페이스
- 현금 IAP (당근 팩)
- HealthKit/Google Fit 자동 연동
- 스마트 알림 타이밍 학습
