## Group 1: 일일 체크인 (Story 3)

### TC-001: 오늘 습관 정상 체크 완료

**대상 스토리**: Story 3
**전제 조건**: 로그인 상태, 오늘 날짜에 해당하는 습관 1개 이상 등록됨

| Step | 액션                        | 기대 결과                                  |
| ---- | --------------------------- | ------------------------------------------ |
| 1    | 메인 화면 진입              | 오늘 날짜에 해당하는 습관 목록 표시        |
| 2    | 습관 항목 탭 (체크)         | 체크 표시 + 토끼 반응 애니메이션 즉시 재생 |
| 3    | 당근 포인트 영역 확인       | +N 팝업 애니메이션 표시 후 잔액 업데이트   |
| 4    | 앱 재시작 후 메인 화면 진입 | 체크 상태 유지                             |

**사후 조건**: 해당 습관 오늘 완료 처리, 스트릭 +1, 당근 포인트 적립
**우선순위**: Critical
