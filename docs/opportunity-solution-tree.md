# Opportunity Solution Tree: Habit Rabbit

**날짜**: 2026-03-07
**프레임워크**: Teresa Torres — Continuous Discovery Habits

---

## Desired Outcome

> **출시 6개월 내 Day-30 리텐션 35% + 유료 전환율 3% 동시 달성**

리텐션 없이는 전환도 없으므로 두 지표를 하나의 목표로 묶음.
리텐션을 먼저 확보하고, 전환은 그 결과로 따라오는 구조.

## O1: 스트릭 깨짐 → 죄책감 → 이탈 (Score: 8.1)

**고객 목소리**: "하루 빠졌더니 스트릭이 사라졌어. 처음부터 다시 시작하기 싫어서 그냥 앱 지웠어."

### 솔루션

| ID   | 솔루션               | 관점     | 설명                                                                  |
| ---- | -------------------- | -------- | --------------------------------------------------------------------- |
| S1-a | **스트릭 복구권**    | PM       | 유료 기능. 월 1개 적립. 하루 실수를 구제 → 유료 전환의 핵심 훅        |
| S1-b | **토끼 위로 플로우** | Designer | 스트릭 깨진 순간 토끼가 "괜찮아, 오늘 다시 시작하면 돼" 감정적 안전망 |
| S1-c | **완료율 대안 지표** | PM       | 스트릭 숫자 대신 "이번 달 달성률 87%" 강조. 올-오어-낫씽 사고 완화    |

**추천 조합**: S1-a (유료 훅) + S1-b (무료 리텐션) 동시 구현

### 실험: E1

- **가설**: 첫 실패 후 복구권 무료 체험 사용자는 미사용자 대비 D7 리텐션 2배 이상
- **방법**: 베타 A/B — 첫 실패 시 복구권 자동 제공 그룹 vs 미제공 그룹
- **측정**: D7 리텐션율
- **성공 기준**: 제공 그룹 D7 리텐션 50% vs 미제공 그룹 25%
- **결정**: 성공 → S1-a를 핵심 유료 업셀 트리거로 확정. 실패 → S1-c(완료율 지표)로 피벗

---

## O2: 트래커 메타습관 실패 → 앱을 잊음 (Score: 7.2)

**고객 목소리**: "설치하고 며칠 쓰다가 그냥... 잊어버렸어. 알림도 무시하게 되더라."

### 솔루션

| ID   | 솔루션                            | 관점     | 설명                                                                 |
| ---- | --------------------------------- | -------- | -------------------------------------------------------------------- |
| S2-a | **홈/잠금화면 위젯**              | Engineer | 앱 안 열어도 한 탭으로 체크. 가장 직접적인 마찰 제거                 |
| S2-b | **토끼 캐릭터 푸시 알림**         | Designer | "당근이 기다리고 있어요" 캐릭터 언어로 알림 차별화                   |
| S2-c | **온보딩 시 "1개부터 시작" 권장** | PM       | 처음 세팅 시 습관 1-2개만 권장. 첫 주 완료율 높임 → 재방문 습관 형성 |

**추천 순서**: S2-c (온보딩 즉시) → S2-b (첫 주 알림 최적화) → S2-a (위젯, E2 결과 후 결정)

### 실험: E2

- **가설**: 습관 1개로 시작한 사용자는 3개로 시작한 사용자보다 D14 리텐션이 높다
- **방법**: 온보딩 분기 — "1개 추천" vs "원하는 만큼" 그룹 비교
- **측정**: D14 리텐션율, 평균 일일 완료율
- **성공 기준**: 1개 그룹 D14 리텐션 40% vs 자유 그룹 20%
- **결정**: 성공 → 온보딩 기본값을 "1개 시작"으로 고정. 실패 → 알림 최적화(S2-b)로 집중

---

## O3: 가치 체감 전 업셀 → 반감 → 이탈 (Score: 7.2)

**고객 목소리**: "쓰다 보니 자꾸 결제하라고 뜨는데, 이게 뭐가 좋은지도 모르겠는데 돈부터 내라고?"

### 솔루션

| ID   | 솔루션                          | 관점     | 설명                                                                  |
| ---- | ------------------------------- | -------- | --------------------------------------------------------------------- |
| S3-a | **복구권 무료 1회 체험 → 업셀** | PM       | 첫 실패 시 복구권 무료 제공 → "유료 플랜에서 매달 받을 수 있어요"     |
| S3-b | **유료 기능 티저 미리보기**     | Designer | 심화 통계/AI 코칭을 블러로 보여주되 "이런 인사이트가 있어요" 미리보기 |
| S3-c | **3개 습관 한도 자연 업셀**     | PM       | 4번째 습관 추가 시도 시 자연스러운 업셀 — 강요 없이 필요성 인식       |

**추천 조합**: S3-c (1차 자연 업셀) + S3-a (감정적 순간 업셀) + S3-b (장기 노출)

### 실험: E3

- **가설**: 블러 처리된 심화 통계 티저를 본 사용자 중 20%는 "더 보기" CTA를 클릭한다
- **방법**: 베타 사용자 심화 통계 화면에 티저 추가 → CTA 클릭률 측정
- **측정**: 티저 CTA 클릭률, 클릭 후 구독 화면 진입률
- **성공 기준**: CTA 클릭률 20% 이상
- **결정**: 성공 → 모든 유료 기능에 티저 적용. 실패 → S3-c(한도 업셀) 단일 전략으로 집중

---

## O4: 목표 과부하 → 초기 이탈 (Score: 6.4)

**고객 목소리**: "처음에 의욕 넘쳐서 6개 습관 등록했는데 일주일 만에 다 실패했어."

**현재 전략으로 부분 해결**: S2-c (온보딩 1개 시작 권장)으로 완화 가능.
추가 솔루션은 O1-O3 검증 후 검토.

---

## O5: 포인트 강박 → 실제 변화 없음 (Score: 4.9)

**고객 목소리**: "당근 모으는 재미로 체크는 하는데, 습관이 진짜 들었는지는 모르겠어."

**전략**: 기능보다 UX 언어 설계로 완화. "당근 획득!" 대신 "오늘도 한 걸음" 같은 습관 형성 중심 카피.
별도 솔루션 불필요 — 카피라이팅/UX 라이팅 가이드라인으로 관리.
